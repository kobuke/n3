# ① N3システム共通利用規約（案）

#### **① N3システム共通利用規約（案）**

本規約は、DMOなんじょう株式会社（以下「当社」）が提供する「N3システム」および「なんじょうデジタル住民ポータル」の利用条件を定めるものです。

* **第1条（目的と実証実験）**  
  1. 本システムは、南城市における観光DX推進を目的とした実証実験（PoC）の一環として運営されます。  
  2. 本実証実験の期間は、原則として2026年3月末日までとします。期間終了後のデータやNFTの取り扱いは、当社の判断により決定されます。  
* **第2条（定義）**  
  1. 「デジタル住民証NFT」：当システムを通じて発行される譲渡不能なトークン（SBT等）を指します。  
  2. 「MORE MORE」：地域への貢献や活動に応じて付与されるコミュニティポイントを指します。  
* **第3条（アカウント連携）**  
  1. ユーザーは、自己のLINEアカウントおよびDiscordアカウントを本システムと連携させるものとします。  
  2. LINEアカウントを紛失、削除、または第三者に奪取されたことにより生じた損害（保有NFTやポイントの消失等）について、当社は一切の責任を負いません。  
* **第4条（禁止事項）**  
  1. システムの脆弱性を突く行為、または不正に「MORE MORE」を取得する行為。  
  2. デジタル住民としての権利を他者に有償で貸与、または不正に譲渡する行為。  
  3. 南城市、他ユーザー、または当社の名誉・信用を毀損する行為。  
* **第5条（免責事項）**  
  1. 当社は、システムの停止、中断、データの消失、およびNFTの市場価値の変動について、いかなる責任も負いません。  
  2. 本サービスは「現状有姿」で提供され、特定の目的への適合性や継続性を保証するものではありません。  
* **第6条（規約の変更）**  
  1. 当社は、実証実験の進捗に応じて、ユーザーの承諾を得ることなく本規約を変更できるものとします。変更後の規約はポータルサイトに掲載した時点で効力を生じます。

# ② プライバシーポリシー（案）

#### **② プライバシーポリシー（案）**

DMOなんじょう株式会社（以下「当社」）は、本プロジェクトにおける利用者の個人情報および関連データの取扱いについて、以下の通り定めます。

* **1\. 取得する情報**  
  1. LINE株式会社より提供される内部識別子（ユーザーID）およびプロフィール情報。  
  2. Discordより提供されるユーザーIDおよびロール（権限）情報。  
  3. 暗号資産ウォレットのアドレスおよび保有するNFTの情報。  
  4. 本システムの利用履歴（ポイント取得・使用履歴、イベント参加履歴等）。  
* **2\. 利用目的**  
  1. デジタル住民証NFTの発行・管理および本人確認のため。  
  2. 保有NFTに応じたDiscord内ロールの自動付与およびコミュニティ運営のため。  
  3. 地域トークン「MORE MORE」の付与および消し込み（使用）管理のため。  
  4. 南城市の観光施策の改善を目的とした統計データの分析。  
* **3\. 第三者への提供および共同利用**  
  1. 本プロジェクトの共同実施体（株式会社NomadResort、日本Web3ツーリズム協会等）に対し、サービス運営に必要な範囲で情報を共有する場合があります。  
  2. 法令に基づく場合を除き、本人の同意なく上記以外の第三者に提供することはありません。  
* **4\. 外部サービスの利用**  
  本システムはLINE、Discord、Shopify等の外部サービスと連携します。各サービスの利用規約およびプライバシーポリシーについては、それぞれの運営会社が定めるものに従うものとします。

# ③ 地域トークン（MORE MORE）運用規定（案）

#### **③ 地域トークン（MORE MORE）運用規定（案）**

本規定は、N3システム内で流通する地域トークン「MORE MORE」（以下「本トークン」）の取扱いを定めるものです。

* **1\. 本トークンの性質**  
  1. 本トークンは、南城市への貢献、交流、および感謝の指標として発行されるコミュニティポイントであり、日本円その他の法定通貨との換金性を持たないものとします。  
  2. 本トークンは、資金決済法における「前払式支払手段」には該当しません。  
* **2\. 付与と取得**  
  1. ユーザーは、当社が指定するイベントへの参加、SNSでの情報発信、アンケート回答等、特定の活動を行うことで本トークンの付与を受けることができます。  
  2. 不正な手段（システムの改ざん等）により取得されたトークンは、当社の判断により即時没収・無効化できるものとします。  
* **3\. 利用および消し込み**  
  1. 本トークンは、当社が指定する特典（体験メニュー、クーポン等）との交換に利用できます。  
  2. 特典との交換が完了（消し込み）した後は、理由の如何を問わずトークンの返還は行えません。  
* **4\. 有効期限と失効**  
  1. 本トークンの有効期限は、原則として2026年3月末日の実証実験終了時までとします。  
  2. LINEアカウントの削除、またはデジタル住民としての資格を喪失した場合、保有するトークンはすべて失効します。

#### **④ Discordコミュニティガイドライン（案）**

本サーバーは、南城市の「デジタル住民」が交流し、共に地域の未来を創るための場所です。参加者は以下のガイドラインを遵守してください。

* **2\. 交流のルール（禁止事項）**  
  * **誹謗中傷・ハラスメント：** 他の参加者や地域住民、特定の団体に対する攻撃的な発言を禁止します。  
  * **不適切な勧誘：** 投資、マルチ商法、宗教、または本プロジェクトに関係のない他NFTプロジェクトへの過度な勧誘行為を禁止します。

* **3\. NFTとロールの運用**  
  * 本サーバーでは「なんじょうデジタル住民証NFT」の保有状況に基づき、自動的に「ロール（役職）」が付与されます。  
  * ロールに応じた限定チャンネルでの情報は、原則として外部への無断転載を禁止します。  
* **4\. モデレーション（管理）**
